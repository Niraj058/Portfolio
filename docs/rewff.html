RewFF Community Platform

An open-source, self-hosted chat environment that experiments with gamified communities, realtime collaboration, and privacy-first social experiences.

Client Personal Project
Role Product Designer & Full-Stack Engineer
Year 2023
Stack React, Node.js, Express, Socket.io, MongoDB

Project Overview

RewFF (short for Rewind & Forward) is a playground for exploring decentralised communities. It combines a React front-end with a Node.js and Socket.io backend to create realtime chat, achievements, and community moderation workflows that can be deployed to any VPS.

Responsibilities

  • Designed the product vision, information architecture, and interface system to support chat, forums, and gamified challenges.
  • Implemented a modular socket layer for private rooms, live presence indicators, and emoji reactions.
  • Built an extensible plugin system so admins can add moderation rules, reward logic, or integrations without touching core code.
  • Authored technical documentation and deployment scripts for Docker-based hosting.

Impact

  • Adopted by small creator communities to host self-governed conversations outside traditional social media.
  • Provided a reference architecture for clients exploring private community platforms and realtime collaboration.
  • Shaped internal reusable modules I now apply to client chatbots, live dashboards, and collaborative tools.
