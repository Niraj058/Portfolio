Dream-o-lytic

An AI-powered companion that transforms dream journals into meaningful insights through natural language processing and symbolic analysis.

Client Personal Project
Role Product Designer & Full-Stack Developer
Year 2025
Stack Laravel, Tailwind CSS, Alpine.js, OpenAI API

Project Overview

Dream-o-lytic invites users to log their nightly dreams, then leverages AI to surface themes, emotions, and symbolism rooted in psychology. The product pairs a serene interface with robust analysis tooling, making reflection and wellness rituals approachable.

Responsibilities

  • Authored the product strategy, user journeys, and visual identity for the MVP release.
  • Implemented authentication, subscription management, and secure data storage in Laravel.
  • Integrated OpenAI endpoints for contextual dream interpretation, sentiment scoring, and key phrase extraction.
  • Built responsive, motion-enhanced UI components with Tailwind CSS and Alpine.js.

Impact

  • Launched a public beta that grew to 1,500+ signups through organic wellness communities.
  • Reduced dream-analysis response time to under 3 seconds via caching and background job queues.
  • Captured anonymized insights to inform future coaching modules and content partnerships.
