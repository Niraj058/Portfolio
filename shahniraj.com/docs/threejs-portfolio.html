Interactive ThreeJS Portfolio

A self-directed portfolio that blends Three.js driven motion, layered storytelling, and instant content updates to introduce my capabilities to prospective partners.

Client Personal Project
Role Creative Developer
Year 2023
Stack Three.js, React, GSAP, Framer Motion

Project Overview

The portfolio was designed as a living product demo. It immerses visitors in a responsive 3D environment backed by structured content, showing the kind of experience and polish I bring to client engagements.

Responsibilities

  • Architected a modular React codebase with a content layer that surfaces case studies and metrics dynamically.
  • Crafted a cinematic hero experience using Three.js particle systems, scroll-tied sequencing, and accessibility-first controls.
  • Engineered a resilient build pipeline with image compression, route-based code splitting, and performance monitoring.
  • Documented reusable animation primitives and theming tokens for future sections and microsites.

Impact

  • Supports rapid iteration—new projects can be added through JSON without code changes.
  • Drove increased inbound interest by showcasing both design sensibility and technical execution.
  • Serves as a reference implementation for clients exploring WebGL storytelling or high-end marketing sites.
